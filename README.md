# 🚀 CRM Laravel – TaskFlow

Ein einfaches CRM-System, entwickelt mit **Laravel**, **Blade** und **Tailwind CSS**.
Das Projekt dient als Lern- und Praxisprojekt und wird schrittweise erweitert.

---

## 📌 Funktionen

* 👤 **Kontakte verwalten (CRUD)**
* 💼 **Deals / Verkaufschancen verwalten**
* 📊 **Dashboard mit Kennzahlen**
* 📈 **Analytics (Statistiken & Charts)**
* 🧾 **Aktivitätsverlauf (History)**
* 🔄 **Deal-Pipeline (Kanban Board)**
* 🌍 **Mehrsprachigkeit (DE / EN / RU)**

---

## 🛠️ Technologien

* ⚙️ PHP / Laravel
* 🎨 Blade Templates
* 💨 Tailwind CSS
* ⚡ Vite
* 🗄️ MySQL

---

## 📂 Projektstruktur

```bash
app/
resources/
  ├── views/
  ├── lang/
       ├── en/
       ├── ru/
       ├── de/
routes/
```

---

## 🌐 Internationalisierung (i18n)

Die Übersetzungen sind nach Domänen strukturiert:

* `app.php` – allgemeine Begriffe
* `contacts.php` – Kontakte
* `deals.php` – Deals
* `analytics.php` – Statistiken
* `activity.php` – Aktivitäten


## 🧠 Lernziele

Dieses Projekt hilft beim Lernen von:

* Laravel Grundlagen
* MVC Architektur
* CRUD Operationen
* Datenbank-Beziehungen
* Internationalisierung (i18n)
* Frontend mit Tailwind
